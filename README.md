# Smart Shopping Cart 智能购物车

This repository is for a Smart Shopping Cart project.

本仓库用于记录智能购物车项目的开发过程、设计思路和后续技术实现。目前项目还处于早期阶段，后续会逐步加入硬件测试、BLE 通信、传感器数据处理和系统原型代码。

---

## Project Goal 项目目标

The goal of this project is to build a smart shopping cart prototype that can collect shopping cart status data and support future functions such as item tracking, automatic checkout, and smart shopping assistance.

本项目目标是开发一个智能购物车原型系统，使购物车能够获取状态信息，并为后续的商品识别、自动结算、购物辅助等功能打下基础。
